## Biggest themes

1. **Public backend trust boundaries are too loose.** The branch still exposes read and write surfaces that let outside callers observe or forge game-visible state.
2. **Runner/orchestration liveness depends too much on “promises always settle.”** Hung RPCs, stale ack files, and catch-up gaps can wedge or desynchronize the automation loop.
3. **Some tests are weaker than they look.** A few key suites appear to validate behavior while actually encoding impossible states, self-fulfilling expectations, or missing negative paths.
4. **Docs/env/package contracts drift from runtime reality.** The clearest examples are `INDEXER_SECRET` and `CLANWORLD_USE_REAL_INDEXER` gaps, frontend stub docs that do not match bootstrap behavior, iNFT/AXL naming drift, and the chain client’s dangerous contract-address fallback.
